# robosys_LED
2020 robot system homework 1

---
このリポジトリは2021年度ロボットシステム学課題1のデバイスドライバーです。
<br>
1の入力時に点灯し、0の入力時に消灯することができます。

---
## 動作環境

### OS : ubuntu 20.04 server
### Raspberry Pi 4 Model B
---
## 使用したもの

・Raspberry Pi4 model B x1

・LED x1

・ブレッドボード x1

・ジャンパワイヤ x4

・抵抗 x1
